# NodeJS REST API
This is me trying to learn some NodeJS.

## Requirements
- [NodeJS](https://nodejs.org/en/download/)
- [Docker](https://www.docker.com/products/docker-desktop)

## Build image, set up network and open ports
```bash
$ docker-compose build --no-cache
```
## Start database
```bash
$ docker-compose up
```

## Start webserver
```bash
$ npm start
```
